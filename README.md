SCU-Wifi-Client 四川大学校园网一键认证客户端

如果您觉得好用的话，请为我点亮⭐⭐哦！感谢您的支持！

1.本工具专为四川大学校园网认证登录开发，替代浏览器网页登录，解决每次上网打开浏览器认证、下线操作繁琐等痛点。

2.支持后台托盘常驻、一键登录/下线、自动记忆账号密码、实时显示网络状态，全程轻量化低内存占用。

Notes：

1.下载exe，双击后运行，首次打开需要填写账号密码，后续会将账号密码存于本机的config.ini文件（请勿删除，该文件仅存于本机，不会泄漏）

2.运行后会生成锁文件，确保系统仅同时运行一个认证客户端，退出软件后自动消失。
